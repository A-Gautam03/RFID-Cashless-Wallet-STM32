RFID-Based Cashless Multipurpose Wallet

A smart cashless payment system developed using STM32 and RFID technology for secure and efficient campus transactions. The system enables students to make contactless payments, manage account balances, view transaction history, and receive low-balance alerts through an interactive LCD interface.

Features:
RFID-based user authentication
Cashless payment processing
Balance management and top-up functionality
Transaction history tracking
Multi-vendor support
Low-balance alerts
LCD-based user interface
Audio feedback using buzzer

Hardware Used:
STM32F303 Nucleo Board
MFRC522 RFID Reader
16×2 LCD Display
Buzzer
Push Buttons

Technologies:
Embedded C
STM32CubeIDE
SPI Communication
RFID Technology

Applications:
Campus Wallet Systems
Smart Cafeteria Payments
Cashless Retail Transactions
Student Identification & Access Systems
